## Style Guidelines:

- Background color: White (#FFFFFF)
- Primary color: Blue (#4285F4) for main elements and actions.
- Secondary color: Red (#DB4437) for important alerts and calls to action.
- Accent color: Yellow (#F4B400) to highlight interactive elements.
- Support Color: Green (#0F9D58) for positive confirmations.
- Body and headline font: 'PT Sans', a humanist sans-serif with a modern feel. Note: currently only Google Fonts are supported.
- Use minimalist, geometric icons throughout the app.
- Use a grid-based layout with plenty of white space for a clean, modern aesthetic.
- Subtle animations and transitions for a smooth user experience.
